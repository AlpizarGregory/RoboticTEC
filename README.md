# How to use it
- Load the code to the Arduino (check `example.ino`) and disconnect it.
- Go to `custom_driver` directory and follow README instructions.
- If needed, install MPI:

```
$ sudo apt install mpich
```
- Compile `cluster.c`

```
$ make
```

- Execute the program

```
$ make run
```
